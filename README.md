# Personal Portfolio Website

This is a simple and responsive **portfolio website / digital CV** built with **HTML, CSS, and JavaScript**.

It showcases my professional profile as a **DevOps Engineer**, including my skills, experience, education, and contact information.

## Features

- Clean and responsive design
- Professional digital CV layout
- Dynamic skills section using JavaScript
- Interactive button feature
- Easy to customize and deploy

## Technologies Used

- HTML5
- CSS3
- JavaScript
- AWS EC2 (for deployment)
- Git & GitHub

- **Contact Information**
- **Skills**

- **Interactive Feature**
